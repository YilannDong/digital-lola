# Pixll 🌱

**Build a little friend for your desktop.**

Pixll lets you design your own digital pet — pick a body, ears, eyes, mouth,
accessory, and colors — then send it to live quietly in the corner of your
screen. It floats, it reacts when you tap it, and it gives you gentle nudges
(drink water, stretch, and a soft "it's getting late" reminder at night).

Cross-platform (macOS + Windows) via Electron.

---

## What it does

- **Lola, your own character** — the exact SVG artwork you provide, embedded
  verbatim (`src/shared/lola.js`) so the desktop pet is pixel-identical to your
  file. Name her and send her to the desktop; the background is transparent so
  she floats cleanly in a corner.

To swap in a different character, drop a new `lola.svg` in `~/Downloads` and run
`node tools/embed-lola.cjs` to re-bake it.
- **Floating desktop companion** — frameless, transparent, always-on-top,
  draggable, remembers where you left it.
- **Quiet interaction** — a slow breathing/float idle motion; tap to get a
  short warm one-liner (time-aware: morning / afternoon / evening / night).
- **Gentle reminders** — occasional low-key prompts on a jittered timer, with a
  wind-down nudge during late hours. Toggle on/off from the right-click menu.

Everything is local. No accounts, no network, no tracking.

## Project structure

```
Pixll/
├── package.json            # electron dependency + start script
├── README.md
└── src/
    ├── main.js             # Electron main: windows, persistence, context menu, IPC
    ├── preload.js          # safe bridge (window.pixll.*) between main and renderers
    ├── shared/             # loaded in BOTH windows as plain browser globals
    │   ├── parts.js        # part catalog + palettes + default pet config
    │   ├── pet-render.js   # renderSVG(config) -> transparent-background pet SVG
    │   └── messages.js     # tap lines + reminders + late-night nudges
    └── renderer/
        ├── builder.html/css/js   # the "make your own pet" screen
        └── pet.html/css/js       # the floating desktop pet
```

The character is the user's own SVG, embedded verbatim in `src/shared/lola.js`
(generated by `tools/embed-lola.cjs`). It renders with a transparent background.

To proof the art, `tools/snap-one.js` renders the character to `tools/one.png`
on a checkerboard background (`npx electron tools/snap-one.js`), and
`tools/snap-builder.js` captures the builder UI — handy for verifying without
launching the whole app.

Saved data lives in one JSON file in the OS user-data dir
(`~/Library/Application Support/pixll/pixll-config.json` on macOS): your pet
design, its last screen position, and reminder settings.

## Install

Requires Node.js 18+ (developed on Node 24). From the project folder:

```bash
npm install
```

If Electron's binary download was skipped by a script-blocking npm config, run:

```bash
node node_modules/electron/install.js
```

---

## Run

```bash
npm start
```

- **First launch** opens the **builder**. Design your pet and click
  **"Bring to desktop →"**.
- After that, launching goes straight to the **floating pet**.
- **Tap** the pet for a line. **Drag** it anywhere. **Right-click** it for
  *Edit my pet…*, *Gentle reminders* (toggle), and *Quit*.

## Packaging (later)

The MVP runs from source. To make a distributable app, add
[`electron-builder`](https://www.electron.build/):

```bash
npm install --save-dev electron-builder
```

Then add to `package.json`:

```json
"scripts": { "dist": "electron-builder" },
"build": {
  "appId": "com.pixll.app",
  "mac": { "target": "dmg" },
  "win": { "target": "nsis" }
}
```

and run `npm run dist`. (Not needed to use or develop the app.)
