# 🛠️ Building 100 Tools with Vibe Coding

A product designer's journey building **100 small tools** by directing AI — one at a time, one reel each. I make the product calls; the AI writes the code.

> **Why:** to prove you don't need to be an engineer to ship real software — you need taste, direction, and a good feedback loop.

## How each one is made

1. **Build** the tool with Claude Code.
2. **Run & record** it locally (screen capture).
3. **Write** the story — a README one-pager, the reel script, and film notes.
4. **Film & edit** the 60-sec reel.
5. **Log** it here (one new row) + link the reel.
